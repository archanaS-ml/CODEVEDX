#Project 1:

# Utility Usage Prediction Tool

A console-based machine learning application that predicts a household's next month utility usage (like electricity or water) based on household size and previous consumption. It combines a menu-driven interface, CSV-based data storage, and a Linear Regression model to turn simple historical usage data into a working prediction system.

This was built as **Project 1** of my AI/ML Engineering Internship at **CodeVedX**, focused on Python fundamentals, ML foundations, and real-world coding workflow practices — including data handling, exception handling, and clean, modular code structure.

#Project 2:

# Student Performance Prediction System

A machine learning application that predicts a student's final academic performance (Pass/Fail) based on attendance, marks, and study hours. It combines data cleaning, exploratory data analysis, and a Logistic Regression model to turn raw student records into meaningful predictions, complete with visual charts and accuracy evaluation.

This was built as **Project 2** of my AI/ML Engineering Internship at **CodeVedX**, focused on data preprocessing, exploratory data analysis, and ML workflow practices — including handling missing values, feature selection, model evaluation, and clean, modular code structure.
